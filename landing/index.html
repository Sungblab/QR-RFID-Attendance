강력한 기능

QR-RFID 출결 시스템이 제공하는 혁신적인 기능들

RFID 카드 출석

학생증에 내장된 RFID 칩으로 리더기에 태그하면 즉시 출석 처리

  • 비접촉 방식으로 위생적
  • 0.1초 이내 인식
  • 카드 분실 시 즉시 재발급

QR 코드 스캔

개인별 고유 QR 코드로 스마트폰만 있으면 언제든 출석 가능

  • 실시간 QR 코드 생성
  • 보안 암호화 적용
  • 모바일 친화적

실시간 대시보드

출석 현황을 한눈에 파악할 수 있는 직관적인 관리자 화면

  • 실시간 출석률 확인
  • 학급별/개인별 통계
  • 시각화 차트 제공

Excel 내보내기

출석 데이터를 Excel 파일로 다운로드하여 추가 분석 가능

  • 기간별 필터링
  • 맞춤형 리포트 생성
  • 자동 서식 적용

알림 시스템

지각, 결석 등 중요 이벤트 발생 시 즉시 알림

  • 실시간 푸시 알림
  • 이메일 알림 설정
  • 맞춤형 알림 규칙

보안 & 개인정보

철저한 보안으로 학생 정보를 안전하게 보호

  • JWT 토큰 인증

  • 접근 권한 관리
